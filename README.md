## рџЋІ Draw Engine Logic

```ts
// Random вЂ” 5 distinct numbers between 1 and 45
randomDraw()

// Algorithmic вЂ” weighted toward most-frequent user scores (+1 base so all numbers possible)
algorithmicDraw(allScores)

// Match users' last-5 score sets against winning numbers
computeMatches(winning, entries) // в†’ { match_5, match_4, match_3 }
```

Pool split: **Match-5: 40%**, **Match-4: 35%**, **Match-3: 25%**.
If no Match-5 winners, that share **rolls over** to next month.
